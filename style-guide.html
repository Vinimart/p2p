Picture to People Style Guide

Paleta de Cores

Cores Principais:

.color-1st

#FF8000

Esta é a cor principal do layout do site, ela está presente em todas as páginas e é utilizada apenas nos elementos de interface, ela nunca deverá ser utilizada como cor de texto. Seu uso se limita à apenas elementos como barras de título de componentes, background de headers, ícones e botões em estado hover.

.color-2nd

#A62900

Esta é a cor secundária do layout do site, além de compor o tema principal do site em conjunto com a cor principal. Seu papel é destacar os elementos de interface como ícones, botões e componentes, para a seleção dos componentes é exclusivamente utilizado a cor secundária. Além de compor elementos de interface essa cor é utilizada nos links em estado hover.

.color-dark

#202020

Este é o preto padrão do site, é a cor padrão dos textos do site e também é a cor base para as sombras e overlays (.color-dark + alpha transparency) Ex: rgba(32, 32, 32, 0.7) ou #202020B3 - O valor de transparencia pode variar no caso das sombras conforme necessário, porém o valor para overlays é sempre 0.7. Nota: Por uma questão de compatibilidade entre browsers foi escolhido o formato RGBA pois o formato Hex + Alpha não é amplamente implementado pelos browsers).

.color-bright

#FFFFFF

Esta é a cor branca padrão do site, seu uso no layout do site se restringe ao background do container principal das páginas. Porém seu uso mais recorrente é na cor da fonte, dos ícones e bordas de elementos que utilizam no background: .color-1st, .color-2nd e .color-dark-grey. Pode ser utilizada também no estado hover de alguns elelentos de interface como o botão Design Tools e nos itens do Footer.

 

Cores Adicionais:

.color-3rd

#E9C4A9

Esta cor é uma variação mais clara da cor principal do site. É utilizada na cor da fonte dos itens do Footer e também é a cor do elemento hr (Ruler).

.color-4th

#F8F0E8

Esta cor é uma variação mais clara da cor .color-dark-grey. É utilizada para compor elementos de interface como background de elementos box [[*Suggestions*]] e background em estado hover dos itens do menu Design Tools.

.color-link

#00528C

Esta é a cor padrão dos elementos âncora da página. É utilizada na maioria dos links e deve ser sempre acompanhado da cor secundária da página em estado hover.

.color-bright-grey

#F5F5F5

Esta cor é o cinza mais claro da escala de cinza da página. Seu uso se restringe à compor o background dos elementos "box" que servem para destacar um container de um background de cor .color-bright. Alguns exemplos são: Container de configuração de efeito, background da sessão Popular Graphic Design Apps na Landing Page e nas páginas auxiliares, e também compõe o background color do header das modais.

.color-gray

#CCCCCC

Esta cor é o cinza médio da página. Seu uso é quase exclusivo para cores de bordas de diversos elementos da página, como os componentes que ficam no formulário porém essa cor também é utilizada como hover das imagens de componente de Wallpaper Generator, 3D Banner Maker e Photo Effect Multilist.

.color-dark-grey

#555555

Esta cor é o cinza mais escuro da escala de cinza da página. O principal uso desta cor são os textos com fonte mais clara, ideal para parágrafos e subtítulos. Também é utilizada para compor a cor da barra de título do container de configuração de efeitos.
